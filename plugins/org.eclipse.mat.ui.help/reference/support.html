Support

Eclipse Memory Analyzer is an open source project. See https://eclipse.dev/mat/support/ for suggestions about obtaining help if you have problems running the tool.

Error Log

For initial investigation of problems with Eclipse Memory Analyzer, consult the error log using Window > Error Log

Acquire Diagnostics

To further assist in diagnosing a problem with Eclipse Memory Analyzer itself try the Acquire diagnostics command, also available from Help > Memory Analyzer Diagnostics... This can capture a thread dump, showing what each Java thread in the Eclipse Memory Analyzer process is doing. It can also capture a heap dump.
