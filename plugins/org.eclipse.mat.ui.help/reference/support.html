Support

Eclipse Memory Analyzer is an open source project. See https://www.eclipse.org/mat/support/ for suggestions about obtaining help if you have problems running the tool.

To assist diagnosing a problem with Eclipse Memory Analyzer itself try the Acquire diagnostics command, also available from Help > Memory Analyzer diagnostics... This can capture a thread dump, showing what each Java thread in the Eclipse Memory Analyzer process is doing. It can also capture a heap dump.
