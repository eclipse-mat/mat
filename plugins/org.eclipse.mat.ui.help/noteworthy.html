Enhancements and fixes

  • The standalone version comes wiht an updated Eclipse platform version. The required minimum Java version is now Java 21
  • The release contains a number of bugfixes and improvements:
    • Issue 143 Escaping $ references in OQL commands.
    • Issue 158 Fix ClassCastException in ComponentReportQuery.addZeroLengthArrays
    • Issue 145 CVE-2025-5222 in com.ibm.icu_77.1.0 - removed dependency on ICU
    • Issue 161 Fix AssertionFailedException in org.eclipse.jface.viewers.StructuredSelection.
